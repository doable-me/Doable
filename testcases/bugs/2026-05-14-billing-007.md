# BUG-2026-05-14-BILLING-007: /pricing page returns 404 for authenticated users

**Severity:** Medium
**Area:** Web — Pricing page
**URL:** https://dev.doable.me/pricing
**Discovered:** 2026-05-14

## Description
Navigating to /pricing shows a "Page not found" error page with a "Back to Dashboard" button. The public pricing page at /pricing is not implemented or routed.

## Steps to Reproduce
1. Log in as any user
2. Navigate to https://dev.doable.me/pricing

## Expected
- Pricing page showing plan comparison (Free, Pro, Business)

## Actual
- 404 "Page not found" with "The page you are looking for does not exist."

## Impact
- Public-facing pricing page missing; users cannot compare plans without going to billing dashboard
- Marketing links to /pricing will be broken
