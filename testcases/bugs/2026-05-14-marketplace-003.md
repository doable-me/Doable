# BUG-2026-05-14-MARKETPLACE-003: /marketplace/feed.json returns 401 (should be public)

**Severity:** Low
**Area:** Marketplace API — Public feed
**Endpoint:** GET /marketplace/feed.json
**Discovered:** 2026-05-14

## Description
TC-MARKET-LIST-035 expects GET /marketplace/feed.json to be a public, unauthenticated endpoint returning a JSON feed of marketplace listings for RSS/aggregator consumption. The endpoint returns 401 Unauthorized.

## Steps to Reproduce
```
GET https://dev-api.doable.me/marketplace/feed.json
(no Authorization header)
```

## Expected
- 200 with JSON feed of marketplace listings

## Actual
- 401 Unauthorized

## Impact
Public feed unavailable; third-party aggregators and marketing tools cannot consume marketplace data.
