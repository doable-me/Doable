# BUG-GH-002: Editor "Connect GitHub" button redirects to /usage

**Date**: 2026-05-14
**Severity**: High
**Feature**: GitHub — editor UI connect flow
**Environment**: dev (https://dev.doable.me)

## Steps to Reproduce

1. Open a project in the editor
2. Click "Connect GitHub" button in editor toolbar or sidebar

## Expected

Initiates GitHub repo OAuth flow (redirects to GitHub or opens connect modal)

## Actual

Browser redirects to /usage page — incorrect route wired to the button

## Impact

Users cannot connect a GitHub repo from within the editor UI.
