# BUG-2026-05-14-MARKETPLACE-002: /marketplace anonymous access redirects to login instead of returning listings

**Severity:** Medium
**Area:** Marketplace — Anonymous access
**Endpoint:** GET /marketplace (web) and GET https://dev-api.doable.me/marketplace/listings
**Discovered:** 2026-05-14

## Description
TC-MARKET-LIST-001 expects the marketplace to be publicly browsable without authentication. Navigating to https://dev.doable.me/marketplace redirects unauthenticated users to the login page. The API endpoint also returns 401 when called without a token.

## Steps to Reproduce
1. Open browser in incognito / clear session
2. Navigate to https://dev.doable.me/marketplace

OR:
```
GET https://dev-api.doable.me/marketplace/listings
(no Authorization header)
```

## Expected
- Public marketplace listings visible without login
- API returns 200 with paginated listing array

## Actual
- Web: redirected to /login
- API: 401 Unauthorized

## Impact
Marketplace is not publicly discoverable. SEO impact. Users cannot browse before signing up.
