# BUG-2026-05-14-MCP-012 — connector-proxy endpoint (/__doable/connector-proxy) not implemented

**Date:** 2026-05-14
**Env:** dev (dev-api.doable.me)
**Severity:** HIGH
**Status:** OPEN
**Test Case:** TC-INTEG-PROXY

## Description

The connector proxy endpoint documented in TC-INTEG-PROXY (`/__doable/connector-proxy`) returns HTTP 404 on both the API and project-scoped routes.

## Evidence

```
GET https://dev-api.doable.me/__doable/connector-proxy → HTTP 404
GET https://dev-api.doable.me/projects/:id/__doable/connector-proxy → HTTP 404
GET https://dev-api.doable.me/projects/:id/connector-proxy-token → HTTP 404
```

## Expected

The connector proxy should be accessible for integration actions. This is the signed JWT proxy mechanism described in TC-INTEG-PROXY.

## Impact

Integration actions via the connector proxy cannot be tested. The proxy is a critical path for integration tool invocations with OAuth credentials.
