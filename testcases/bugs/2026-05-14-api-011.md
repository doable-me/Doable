# BUG-API-011: GET /admin/users email filter not working

**Date:** 2026-05-14
**Severity:** high
**Area:** TC-API-ADMIN-004

## Summary
`GET /admin/users?email=owner-pro@doable.me` returns all 28 users instead of the filtered 1 user. The `?email=` query parameter is silently ignored.

## Steps to Reproduce
```bash
# Total users
curl "https://dev-api.doable.me/admin/users" -H "Authorization: Bearer $ADMIN_TOKEN"
# Returns 28 users

# Filtered
curl "https://dev-api.doable.me/admin/users?email=owner-pro@doable.me" -H "Authorization: Bearer $ADMIN_TOKEN"
# Also returns 28 users — filter ignored
```

## Expected
1 user matching the email.

## Impact
Admin cannot search users by email. Usability and operational tooling broken.
