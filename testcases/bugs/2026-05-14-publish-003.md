# BUG-2026-05-14-publish-003: Published project dashboard card does not show live URL chip

**Severity:** Medium  
**Environment:** dev (https://dev.doable.me)  
**Date:** 2026-05-14  
**Affected TCs:** TC-PUBLISH-LIFECYCLE-025

## Summary
The dashboard project card for "Smoke Vite Updated" shows a "Published" badge, but does not show a live URL chip. Per test case TC-PUBLISH-LIFECYCLE-025, each published card should show the live URL.

## Steps to Reproduce
1. Log in as owner-pro@doable.me
2. Navigate to /dashboard
3. Observe the "Smoke Vite Updated" project card (status: Published)

## Expected
Card shows Published badge AND a clickable live URL (e.g., https://dev-smoke-vite.doable.me).

## Actual
Card shows "Published" badge only. No live URL chip visible.

## Root Cause (Suspected)
Likely downstream of BUG-002 — since `published_url` is null in the API response, the UI has no URL to display.
