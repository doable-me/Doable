# BUG-API-005: POST /projects/:id/archive and /unarchive return 404

**Date:** 2026-05-14
**Severity:** high
**Area:** TC-API-PROJECTS-049, TC-API-PROJECTS-051

## Summary
Both `POST /projects/:id/archive` and `POST /projects/:id/unarchive` return 404 Not Found. These endpoints are documented in the test spec and routes.ts but are not reachable.

## Steps to Reproduce
```bash
curl -X POST "https://dev-api.doable.me/projects/<pid>/archive" \
  -H "Authorization: Bearer $TOKEN"
# Response: {"error":"Not Found","path":"/projects/<pid>/archive"}

curl -X POST "https://dev-api.doable.me/projects/<pid>/unarchive" \
  -H "Authorization: Bearer $TOKEN"
# Response: {"error":"Not Found","path":"/projects/<pid>/unarchive"}
```

## Expected
200 with `{archived:true}` / `{archived:false}`.

## Impact
Archive/unarchive functionality is completely unavailable via API.
