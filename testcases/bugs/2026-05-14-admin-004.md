# BUG-ADMIN-004: /admin/projects/:id returns 404 — project detail endpoint missing

**Date:** 2026-05-14
**Severity:** High
**Test Cases:** TC-ADMIN-PROJECTS-020, TC-ADMIN-PROJECTS-021, TC-ADMIN-PROJECTS-022

## Summary
GET /admin/projects/:projectId returns 404 "Not Found". The project detail endpoint does not exist. Admins cannot drill into individual project details.

## Steps to Reproduce
```
GET https://dev-api.doable.me/admin/projects/80bd2376-2c67-4df8-b459-ad4e8dce0c81
Authorization: Bearer <admin-token>
```

## Expected
200 with project detail: members, files count, build artifacts, recent audit, dovault state.

## Actual
```json
{"error": "Not Found", "path": "/admin/projects/80bd2376-2c67-4df8-b459-ad4e8dce0c81"}
```

## Impact
Admin cannot view project detail page. Actions like "soft delete", "restore", "open as user", "view activity" are all unavailable.
