# BUG-2026-05-14-MCP-002 — Built-in MCP connector deletable (should be blocked)

**Date:** 2026-05-14
**Env:** dev (dev-api.doable.me)
**Severity:** HIGH
**Status:** OPEN
**Test Case:** TC-MCP-CONNECTOR-013

## Description

Built-in MCP connectors (Markdown Builder, PDF Builder, etc.) can be deleted via DELETE /workspaces/:wsId/connectors/:id, even though they are platform-managed built-ins that should be immutable.

## Evidence

```
DELETE /workspaces/492bdda5-6f66-46c7-a6e6-22a8b391fa9d/connectors/00ee53d6-aa84-4f5f-9c0e-ef39c4c4a292
Response: HTTP 200 {"data":{"id":"00ee53d6-aa84-4f5f-9c0e-ef39c4c4a292","deleted":true}}
```

The Markdown Builder (a platform built-in) was successfully deleted. After deletion, the connector list only shows 3 connectors instead of 4.

## Expected

HTTP 403 or 400 — built-in connectors should not be deletable. Only user-created connectors should be removable.

## Impact

Users can accidentally or intentionally remove the built-in MCP App servers (Markdown Builder, PDF Builder, Presentation Builder, Spreadsheet Builder), breaking MCP Apps functionality for the entire workspace.
