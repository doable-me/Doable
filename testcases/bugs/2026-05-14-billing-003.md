# BUG-2026-05-14-BILLING-003: "Manage Subscription" button navigates to /usage instead of Stripe portal

**Severity:** High
**Area:** Billing UI — Manage Subscription CTA
**URL:** https://dev.doable.me/billing
**User:** owner-pro@doable.me (Pro Plan)
**Discovered:** 2026-05-14

## Description
Clicking the "Manage Subscription" button on the Billing page redirects the user to /usage (the usage dashboard) instead of launching the Stripe billing portal.

## Steps to Reproduce
1. Log in as owner-pro@doable.me
2. Navigate to https://dev.doable.me/billing
3. Click "Manage Subscription" button

## Expected
- POST /billing/portal called
- User redirected to Stripe Customer Portal URL

## Actual
- User redirected to https://dev.doable.me/usage
- No portal URL fetched

## API Evidence
POST /billing/portal returns: `{"error":"No billing account found"}` — the API returns an error, and the UI falls back to /usage instead of showing an error message.
