# BUG-API-007: PUT /workspaces/:wid returns 404

**Date:** 2026-05-14
**Severity:** high
**Area:** TC-API-WS-015, TC-API-WS-016

## Summary
`PUT /workspaces/:wid` returns 404 Not Found for all callers including workspace owner. The endpoint is defined in the test spec but not routed.

## Steps to Reproduce
```bash
curl -X PUT "https://dev-api.doable.me/workspaces/<wid>" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name":"New Name"}'
# Response: {"error":"Not Found","path":"/workspaces/<wid>"}
```

## Expected
200 with updated workspace.

## Impact
Workspace name/slug updates are impossible via API.
