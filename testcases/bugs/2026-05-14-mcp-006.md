# BUG-2026-05-14-MCP-006 — Integration catalog category filter broken

**Date:** 2026-05-14
**Env:** dev (dev-api.doable.me)
**Severity:** MEDIUM
**Status:** OPEN
**Test Case:** TC-INTEG-LIST-004

## Description

The ?category= filter on /integrations/catalog returns 0 results for all categories including "crm".

## Evidence

```
GET /integrations/catalog?category=crm → count=0
GET /integrations/catalog → count=533 (all integrations, no category field filtering)
```

Integration data does include a "category" field (e.g. "marketing", "developer_tools", "finance_payments"), but the filter returns empty results.

## Expected

?category=crm should return CRM integrations (HubSpot, Salesforce, etc.)

## Impact

Category browsing in the integration marketplace is broken.
