# BUG-API-021: Chat session/message endpoints return 404

**Date:** 2026-05-14
**Severity:** high
**Area:** TC-API-CHAT-016, TC-API-CHAT-021

## Summary
`GET /chat/:projectId/sessions` and `GET /chat/:projectId/messages` both return 404. The chat API surface at `/chat/:pid/*` is not routed.

## Steps to Reproduce
```bash
curl "https://dev-api.doable.me/chat/<pid>/sessions" \
  -H "Authorization: Bearer $TOKEN"
# HTTP 404: {"error":"Not Found","path":"/chat/<pid>/sessions"}

curl "https://dev-api.doable.me/chat/<pid>/messages" \
  -H "Authorization: Bearer $TOKEN"
# HTTP 404
```

## Expected
200 with sessions/messages list.

## Impact
Chat history, session management, and message retrieval are all inaccessible via API. The copilot chat feature is crippled.
