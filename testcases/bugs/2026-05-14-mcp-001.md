# BUG-2026-05-14-MCP-001 — dev-api.doable.me crash loop (502 Bad Gateway)

**Date:** 2026-05-14
**Env:** dev (dev-api.doable.me)
**Severity:** CRITICAL (blocks all authenticated testing)
**Status:** OPEN

## Description

The dev API server (dev-api.doable.me) is in a crash loop. The /health endpoint alternates between HTTP 200 (uptime 5-11 seconds) and HTTP 502 Bad Gateway. No authenticated endpoint is reachable.

## Evidence

- `curl https://dev-api.doable.me/health` → HTTP 502 for most requests
- When 200 briefly returns, uptime is only 5-11 seconds (process just restarted)
- Browser console: "CORS policy: No 'Access-Control-Allow-Origin' header" because 502 from Cloudflare has no CORS headers
- Login returns: "Something went wrong. Please try again." in UI
- API login returns: `error code: 502`

## Steps to Reproduce

1. curl https://dev-api.doable.me/health
2. Observe: HTTP 502 repeatedly (intermittent 200 for ~10s when process restarts)
3. All authenticated API calls blocked

## Impact

- All MCP connector tests blocked
- All integration tests blocked
- Login impossible (blocks all browser testing)
- 390 planned test cases cannot be executed
