# BUG-API-017: GET /github/install-url and /github/oauth/start return 404

**Date:** 2026-05-14
**Severity:** high
**Area:** TC-API-GITHUB-001, TC-API-GITHUB-027

## Summary
`GET /github/install-url` and `GET /github/oauth/start` both return 404. These endpoints are required to initiate the GitHub OAuth/App installation flow.

## Steps to Reproduce
```bash
curl "https://dev-api.doable.me/github/install-url" \
  -H "Authorization: Bearer $TOKEN"
# HTTP 404

curl -L "https://dev-api.doable.me/github/oauth/start" \
  -H "Authorization: Bearer $TOKEN"
# HTTP 404
```

## Expected
- `/github/install-url`: 200 with `{url: "https://github.com/apps/..."}` 
- `/github/oauth/start`: 302 redirect to GitHub OAuth

## Impact
GitHub integration cannot be initiated. Users cannot connect GitHub accounts.
