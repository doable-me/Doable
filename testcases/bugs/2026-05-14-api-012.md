# BUG-API-012: Multiple /admin endpoints return 404 (not implemented)

**Date:** 2026-05-14
**Severity:** high
**Area:** TC-API-ADMIN-012, 016, 021, 023, 025, 031, 035, 037, 038

## Summary
The following admin endpoints are documented in routes.ts and test specs but return 404 Not Found:

| Endpoint | TC | Expected |
|---|---|---|
| GET /admin/workspaces | ADMIN-012 | 200 list |
| GET /admin/audit | ADMIN-016 | 200 audit log |
| GET /admin/traces | ADMIN-021 | 200 traces |
| POST /admin/tracing/start | ADMIN-023 | 200 |
| GET /admin/tracing/status | ADMIN-025 | 200 |
| PUT /admin/features/:flag | ADMIN-031 | 200 |
| POST /admin/email/send-test | ADMIN-035 | 200 |
| GET /admin/ops/queue | ADMIN-037 | 200 |
| POST /admin/tools/run/:tool | ADMIN-038 | 200 |

## Steps to Reproduce
```bash
curl "https://dev-api.doable.me/admin/workspaces" -H "Authorization: Bearer $ADMIN_TOKEN"
# {"error":"Not Found"}
# (same for all above)
```

## Impact
Large portions of the admin panel are non-functional. Audit logging, tracing controls, email testing, and operational tooling are all inaccessible.
