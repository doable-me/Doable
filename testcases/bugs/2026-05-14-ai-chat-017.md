# BUG-017: Session management endpoints missing (PATCH session, rename, model switch)

**Date:** 2026-05-14
**Suite:** 05-ai-chat
**Severity:** High
**Test Cases:** TC-AI-CHAT-MODES-004, TC-AI-CHAT-HISTORY-031, TC-AI-CHAT-MODELS-003

## Description
Multiple session management operations return 404 because the PATCH `/projects/:id/chat/session` endpoint does not exist. The following operations are not implemented:
- Mode switching (PATCH mode)
- Session rename (PATCH title)
- Model switching (PATCH model)

## Reproduction
```
PATCH https://dev-api.doable.me/projects/:id/chat/session
{"mode":"plan"}
=> HTTP 404 {"error":"Not Found","path":"/projects/.../chat/session"}

PATCH https://dev-api.doable.me/projects/:id/chat/session
{"title":"My renamed session"}
=> HTTP 404

PATCH https://dev-api.doable.me/projects/:id/chat/session
{"model":"anthropic/claude-3-5-sonnet"}
=> HTTP 404
```

## Expected
PATCH endpoint returns 200 with updated session; subsequent sends use the new mode/model.

## Impact
Users cannot switch modes mid-session, rename sessions, or change the AI model without deleting and recreating. These are core session management features.
