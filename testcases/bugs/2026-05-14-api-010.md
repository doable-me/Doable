# BUG-API-010: GET /workspaces/:wid/billing/plan returns 404

**Date:** 2026-05-14
**Severity:** medium
**Area:** TC-API-WS-041

## Summary
`GET /workspaces/:wid/billing/plan` returns 404. The endpoint is documented in the workspace test spec.

## Steps to Reproduce
```bash
curl "https://dev-api.doable.me/workspaces/<wid>/billing/plan" \
  -H "Authorization: Bearer $TOKEN"
# Response: {"error":"Not Found","path":"/workspaces/<wid>/billing/plan"}
```

## Expected
200 with plan information.

## Note
`GET /billing/plans` (top-level) works (200). The per-workspace billing plan endpoint is missing.
