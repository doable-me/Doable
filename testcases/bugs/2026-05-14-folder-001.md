# BUG-FOLDER-001: Folder cycle detection missing (parentId=self allowed)

**Date**: 2026-05-14
**Severity**: Medium
**Feature**: Folders — cycle/loop prevention
**Environment**: dev (https://dev-api.doable.me)

## Steps to Reproduce

1. Authenticate as owner-pro@doable.me
2. Create folder F1
3. PATCH /folders/{F1.id} with body `{"parentId": "{F1.id}"}` (self-reference)

## Expected

400 Bad Request — cannot set a folder as its own parent (cycle detected)

## Actual

200 OK — folder updated with parentId pointing to itself, creating a cycle

## Impact

Self-referential folders create infinite loops in any recursive tree traversal (sidebar rendering, export, search). Could cause stack overflow or infinite loop in client/server code.
