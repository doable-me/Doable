# BUG-2026-05-14-MARKETPLACE-006: /marketplace/bundles returns 404 (bundles not implemented)

**Severity:** Low
**Area:** Marketplace API — Bundles
**Endpoint:** GET /marketplace/bundles
**Discovered:** 2026-05-14

## Description
TC-MARKET-CATEGORIES-BUNDLES test cases expect bundle endpoints for curated listing collections. All bundle endpoints return 404 Not Found.

## Steps to Reproduce
```
GET https://dev-api.doable.me/marketplace/bundles
Authorization: Bearer {valid_token}
```

Also:
```
GET https://dev-api.doable.me/marketplace/bundles/{id}
POST https://dev-api.doable.me/marketplace/bundles (admin)
```

## Expected
- 200 with list of bundles
- Admin can create/edit bundles

## Actual
- 404 Not Found

## Impact
Bundle/collection feature unavailable; curated groupings of related tools not possible.

TC-MARKET-CATEGORIES-BUNDLES test cases
