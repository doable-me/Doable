# BUG-API-020: GET /domains returns 404 (endpoint not implemented)

**Date:** 2026-05-14
**Severity:** high
**Area:** TC-API-DEPLOY-021, TC-API-DEPLOY-022

## Summary
`GET /domains` and related domain management endpoints (`POST /domains`, `DELETE /domains/:id`, `POST /domains/:id/verify`) all return 404. The custom domain management feature is unavailable via API.

## Steps to Reproduce
```bash
curl "https://dev-api.doable.me/domains" \
  -H "Authorization: Bearer $TOKEN"
# HTTP 404: {"error":"Not Found","path":"/domains"}
```

## Expected
200 with list of user's custom domains.

## Impact
Custom domain management is completely unavailable.
