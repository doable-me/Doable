# BUG-API-009: DELETE /workspaces/:wid returns 200 instead of 204

**Date:** 2026-05-14
**Severity:** low
**Area:** TC-API-WS-019

## Summary
`DELETE /workspaces/:wid` returns HTTP 200 instead of the expected 204 No Content.

## Steps to Reproduce
```bash
curl -si -X DELETE "https://dev-api.doable.me/workspaces/<wid>" \
  -H "Authorization: Bearer $TOKEN"
# Response: HTTP/1.1 200 OK
```

## Expected
HTTP/1.1 204 No Content

## Impact
Minor REST convention violation consistent with DELETE /projects returning 200.
