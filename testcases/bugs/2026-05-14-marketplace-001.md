# BUG-2026-05-14-MARKETPLACE-001: POST /marketplace/install returns 500 for nonexistent listing

**Severity:** Medium
**Area:** Marketplace API — Install endpoint
**Endpoint:** POST /marketplace/install
**Discovered:** 2026-05-14

## Description
TC-MARKET-INSTALL-001 expects a 404 when installing a nonexistent listing ID. The endpoint returns 500 Internal Server Error instead, leaking server error without proper not-found handling.

## Steps to Reproduce
```
POST https://dev-api.doable.me/marketplace/install
Authorization: Bearer {valid_token}
Content-Type: application/json
Body: {"listingId":"00000000-0000-0000-0000-000000000000","workspaceId":"492bdda5-6f66-46c7-a6e6-22a8b391fa9d"}
```

## Expected
- 404 Not Found with descriptive error message

## Actual
- 500 Internal Server Error

## Impact
Unhandled database errors exposed to clients; no graceful not-found handling for install endpoint.
