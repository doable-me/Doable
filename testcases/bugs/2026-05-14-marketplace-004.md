# BUG-2026-05-14-MARKETPLACE-004: POST /marketplace/reviews returns 404 (endpoint not implemented)

**Severity:** Medium
**Area:** Marketplace API — Reviews
**Endpoint:** POST /marketplace/reviews
**Discovered:** 2026-05-14

## Description
TC-MARKET-REVIEW test cases expect a reviews endpoint for creating, reading, and managing listing reviews. All review endpoints return 404 Not Found, indicating the feature is not implemented.

## Steps to Reproduce
```
POST https://dev-api.doable.me/marketplace/reviews
Authorization: Bearer {valid_token}
Content-Type: application/json
Body: {"listingId":"...","rating":5,"comment":"Great tool"}
```

Also:
```
GET https://dev-api.doable.me/marketplace/listings/{id}/reviews
```

## Expected
- 201 Created with review object
- GET returns paginated reviews

## Actual
- 404 Not Found on all /marketplace/reviews endpoints

## Impact
Users cannot leave reviews; listing quality signaling not available; trust/social proof missing.
