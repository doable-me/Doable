# BUG-2026-05-14-MARKETPLACE-005: POST /marketplace/reports returns 404 (moderation not implemented)

**Severity:** Medium
**Area:** Marketplace API — Moderation/Reports
**Endpoint:** POST /marketplace/reports
**Discovered:** 2026-05-14

## Description
TC-MARKET-MODERATION test cases expect endpoints for reporting listings and admin moderation actions. All moderation endpoints return 404 Not Found, indicating the feature is not implemented.

## Steps to Reproduce
```
POST https://dev-api.doable.me/marketplace/reports
Authorization: Bearer {valid_token}
Content-Type: application/json
Body: {"listingId":"...","reason":"spam"}
```

Also:
```
GET https://dev-api.doable.me/marketplace/reports (admin)
PATCH https://dev-api.doable.me/marketplace/listings/{id}/status (admin)
```

## Expected
- 201 Created for report submission
- Admin endpoints return report list and allow status changes

## Actual
- 404 Not Found on all /marketplace/reports and moderation endpoints

## Impact
No content moderation possible; malicious or low-quality listings cannot be flagged or removed.
