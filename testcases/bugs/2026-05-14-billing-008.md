# BUG-2026-05-14-BILLING-008: Enterprise plan missing from /billing/plans response

**Severity:** Medium
**Area:** Billing API — Plans list
**Endpoint:** GET /billing/plans
**Discovered:** 2026-05-14

## Description
TC-BILLING-PLANS-005 expects an enterprise plan with `contactSales=true` and price omitted. The GET /billing/plans endpoint returns only 3 plans: free, pro, business. Enterprise is absent.

## Steps to Reproduce
```
GET https://dev-api.doable.me/billing/plans
```

## Expected
- 4 plans including enterprise with contactSales=true

## Actual
- 3 plans: free, pro, business
- No enterprise plan
- No `contactSales` field on any plan

## Also Missing from Plan Schema
- `storageMb` field not present (TC-BILLING-PLANS-006)
- `priceCents` field not present (uses `priceMonthly` in dollars instead)
- `interval` field not present
