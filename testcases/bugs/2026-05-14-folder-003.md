# BUG-FOLDER-003: Folder color field not persisted (schema gap)

**Date**: 2026-05-14
**Severity**: Low
**Feature**: Folders — color property
**Environment**: dev (https://dev-api.doable.me)

## Steps to Reproduce

1. Authenticate as owner-pro@doable.me
2. POST /folders with body `{"name": "Colored Folder", "workspaceId": "...", "color": "#FF5733"}`

## Expected

201 Created with `color: "#FF5733"` in the response

## Actual

201 Created but response does not include a `color` field — the property is silently ignored

## Impact

UI cannot persist folder colors. Any color-coding feature in the frontend has no backend support.
