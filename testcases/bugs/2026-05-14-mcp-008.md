# BUG-2026-05-14-MCP-008 — listTools returns HTTP 500 for inactive HTTP connector

**Date:** 2026-05-14
**Env:** dev (dev-api.doable.me)
**Severity:** HIGH
**Status:** OPEN
**Test Case:** TC-MCP-CONNECTOR-022

## Description

GET /workspaces/:wsId/connectors/:id/tools returns HTTP 500 when the connector is an HTTP-type connector with status "inactive" (never connected).

## Evidence

```
GET /workspaces/c1083c8a.../connectors/4826f3ae.../tools
(connector: transportType=streamable_http, status=inactive, serverUrl=https://example.com/mcp)
Response: HTTP 500
```

Markdown Builder (stdio, status=active) returns HTTP 200 with tools correctly.

## Expected

HTTP 200 with empty tools array OR a structured error (503 "connector inactive") — not an unhandled 500.

## Impact

Internal server error exposed to users when querying tools on an inactive HTTP connector.
