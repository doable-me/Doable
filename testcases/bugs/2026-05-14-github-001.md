# BUG-GH-001: /auth/github/repo/start returns 500

**Date**: 2026-05-14
**Severity**: High
**Feature**: GitHub OAuth — repo connect flow
**Environment**: dev (https://dev-api.doable.me)

## Steps to Reproduce

1. Authenticate as owner-pro@doable.me
2. GET https://dev-api.doable.me/auth/github/repo/start (with valid Bearer token)

## Expected

302 redirect to GitHub OAuth authorization with scope=repo

## Actual

500 Internal Server Error

## Also Affected

- Unauthenticated request to /auth/github/repo/start → 500 (should be 401)

## Impact

GitHub repo connect flow completely broken. Users cannot connect GitHub repositories to projects.

## Test Cases

TC-GH-OAUTH-015, TC-GH-OAUTH-016
