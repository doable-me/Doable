# BUG-API-006: POST /projects/:id/share returns 404

**Date:** 2026-05-14
**Severity:** high
**Area:** TC-API-PROJECTS-072, TC-API-PROJECTS-073

## Summary
`POST /projects/:id/share` returns 404 Not Found. The endpoint is documented but not implemented/routed.

## Steps to Reproduce
```bash
curl -X POST "https://dev-api.doable.me/projects/<pid>/share" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"visibility":"public"}'
# Response: {"error":"Not Found","path":"/projects/<pid>/share"}
```

## Expected
200 with share link, or 400 for invalid visibility.

## Impact
Project sharing functionality unavailable.
