# BUG-API-008: POST /workspaces/:wid/members returns 404

**Date:** 2026-05-14
**Severity:** high
**Area:** TC-API-WS-025, TC-API-WS-026

## Summary
`POST /workspaces/:wid/members` (invite member) returns 404 Not Found. The endpoint is documented but not reachable.

## Steps to Reproduce
```bash
curl -X POST "https://dev-api.doable.me/workspaces/<wid>/members" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","role":"editor"}'
# Response: {"error":"Not Found","path":"/workspaces/<wid>/members"}
```

## Expected
201 with invite created.

## Impact
Workspace member invitations unavailable via API. Teams cannot grow.
